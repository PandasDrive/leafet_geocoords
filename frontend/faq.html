Frequently Asked Questions

What are .fbf and .sff files?

They are custom binary formats used to store signal data containing geographical coordinates.

What is hex data?

It's a representation of the binary data from the files. You can copy the content of a file as a hex string and paste it here. Ensure it is not an odd number of Hex Fields

How much data can I process at once?

The application can handle reasonably sized files and hex strings, but performance may degrade with very large inputs Not recommend over 1000 lines of hex fields.

What signals are supported?

Currently, the application supports XXX formats that include geographical coordinates. More formats may be added in future updates.

Why do I need a local tile server for the map?

To use the application completely offline. The application is configured to look for map tiles locally. You can obtain map tiles from sources like OpenStreetMap and host them with a simple local server.
