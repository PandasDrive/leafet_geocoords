About GeoSignal Parser

This tool is designed to parse and visualize geographical data from custom Hexadecminal signal formats (.fbf, .sff).

It can process files or raw hexadecimal strings to extract coordinates and display them on an interactive map.

The application is built using Flask for the backend and Leaflet.js for map rendering, ensuring a smooth and responsive user experience.

For more information, visit the FAQ page.

Developed by Erin Lord (emlord) / Base idea from Lauren and Jade.

Source code available on GitLab.
